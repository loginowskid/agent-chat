Embedded Chat Example

The chat widget below is embedded directly into this page using a mount target. It uses the light theme and connects to a local server.

How It Works

Embedded mode — mount into a container:

<div id="my-chat" style="height: 500px;"></div>
<script src="agent-chat.js"
    data-mount="#my-chat"
    data-server="http://localhost:8211"
    data-theme="light">
</script>

Floating bubble mode — just include the script:

<script src="agent-chat.js" data-server="http://localhost:8211"></script>

Features

Click/tap a message to select it. User messages show a ✏️ edit icon. Assistant messages show a ↩️ reply icon. Right-click any message to create a topic or assign it to an existing one.

Topics group related messages into collapsible sections. Archive a topic by clicking the ✓ on its header. Toggle archived topics with the 📋 button.
